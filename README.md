# Pokedex


## Run on dev

1. Clone repository
2. Run
```
yarn install
```

3. It's required Nest CLI is installed
```
npm i -g @nestjs/cli
```

4. Run database
```
docker-compose up -d
```

## Used Stack
* MongoDB
* Nest
